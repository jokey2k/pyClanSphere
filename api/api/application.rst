Core Application
================

Most of the public members are exported to the :mod:`pyClanSphere.api` module, which you can asterisk import safely.

If you develop a module, the *add_* functions of :class:`pyClanSphere.application.pyClanSphere` might be of interest, as they can be called during plugin setup.
