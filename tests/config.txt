Basic transaction test. Also tests that default values are not written to the
config file.

	>>> t = app.cfg.edit()
	>>> t['clan_title']
	iu'My Clan Page'
	>>> t['clan_title'] = 'Another Site Title'
	>>> t.commit()
	>>> app.cfg['clan_title']
	u'Another Site Title'


Test revert_to_default

	>>> t = app.cfg.edit()
	>>> t.revert_to_default('clan_title')
	>>> t['clan_title']
	iu'My Clan Page'
	>>> t.commit()
	>>> app.cfg['clan_title']
	iu'My Clan Page'


Transactions can only be committed once:
